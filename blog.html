Blog Menggunakan Bootstrap

Bootstrap memang merupakan istilah yang berasal dari bahasa Inggris dalam dunia pengembangan web, dan belum ada padanan resmi dalam Bahasa Indonesia. Istilah ini merujuk pada sebuah framework CSS populer yang digunakan untuk mendesain website dengan cepat dan mudah.

Bootstrap menyediakan berbagai kelas CSS yang dapat digunakan untuk mendesain elemen HTML dengan mudah. Anda dapat menggunakan kelas-kelas ini untuk membuat berbagai komponen web seperti: Tombol; Formulir; Navigasi; Tabel; dan banyak lagi.
